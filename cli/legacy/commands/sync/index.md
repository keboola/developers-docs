**Synchronization between a [local directory](/cli/legacy/structure/) and a [project](/cli/legacy/#subsystems).**

```
kbc sync [command]
```

|---
| Command | Description
|-|-|-
| [kbc sync init](/cli/legacy/commands/sync/init/) | Initialize a new local directory and run `kbc sync pull`. |
| [kbc sync pull](/cli/legacy/commands/sync/pull/) | Sync a project to the local directory. |
| [kbc sync push](/cli/legacy/commands/sync/push/) | Sync the local directory to the project. |
| [kbc sync diff](/cli/legacy/commands/sync/diff/) | Show differences between the local directory and the project. |
